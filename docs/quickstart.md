# Quick Start

## Prerequisites

1. OpenCode
2. Node.js 18+
3. Python 3.10+ with `uv`
4. Optional `bun` for plugin rebuilds
5. Obsidian Vault at `/Users/xuyongheng/Obsidian-Vault`
6. Zotero desktop if you use Zotero MCP

## Start

```bash
cd /Users/xuyongheng/PhD-Research
opencode
```

## Verify

```bash
bash .opencode/verifiers/run-all.sh
```
