# Creating Domain Packs

Domain knowledge in this project is currently centered on `domains/ai-in-education/`.

If you later expand to another field, create a parallel domain pack manually with the same structure.

## Required files

- `domain.yaml`
- `theories.yaml`
- `methods.yaml`
- `topics.yaml`
- `social-issues.yaml`
- `journals.md`
- `keyword-mapping.md`
